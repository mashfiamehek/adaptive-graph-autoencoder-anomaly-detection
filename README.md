# Adaptive Graph Autoencoder for Anomaly Detection

An Adaptive Graph Autoencoder framework for anomaly detection in Industrial Internet of Things (IIoT) systems using Graph Neural Networks and PyTorch.

## Project Overview

Industrial IoT environments generate massive volumes of interconnected sensor data. This project models industrial systems as graphs and applies an adaptive graph autoencoder to identify abnormal machine behavior through reconstruction-based anomaly detection.

## Key Features

- Graph-based anomaly detection framework
- Adaptive GCN-LSTM Autoencoder architecture
- Correlation-based sensor graph construction
- Reconstruction error based anomaly scoring
- Performance evaluation using Accuracy, Precision, Recall, F1-score and ROC-AUC
